## 리스트

### 순서 없는 리스트

```*```, ```+```, ```-```

* 부리또
* 돈가스
  + 치킨
  + 피자
    - 국밥
    - 라면

### 순서 있는 리스트

1, 2, 3, ...

들여쓰기하면 모양 바뀜

1. 부리또
2. 돈가스
   1. 치킨
   2. 피자
      1. 국밥
      2. 라면
